# Recruiter Pitch

## Short Pitch

Retail Intelligence Platform is an end-to-end analytics engineering project
that turns raw retail CSV files into a PostgreSQL warehouse, Power BI-ready
marts, a FastAPI service, and a baseline sales forecasting workflow.

## What It Shows

- I can clean and validate messy real-world datasets with Python.
- I can design PostgreSQL staging, warehouse, and mart layers.
- I understand dimensional modeling and source-system limitations.
- I can add dbt tests and documentation without overengineering.
- I can expose analytics through a small FastAPI service.
- I can prepare ML-ready data and train a baseline forecasting model.
- I can document a project clearly for technical and non-technical reviewers.

## Interview Talking Points

- The warehouse keeps Online Retail and Walmart separate where the source data
  does not support reliable joins.
- The marts are designed for BI consumption, so dashboard logic stays simple.
- The ML workflow starts with a transparent baseline before adding complexity.
- The project uses `.env`, `.gitignore`, and Docker to avoid committing secrets
  or large datasets.

## Role Fit

This project is relevant for:

- Junior Data Analyst
- BI Developer
- Analytics Engineer Intern
- Junior Data Engineer
- Data/AI Analyst
