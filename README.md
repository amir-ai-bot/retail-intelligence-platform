# Retail Intelligence Platform

## Overview

Retail Intelligence Platform is an end-to-end analytics project built to transform raw retail datasets into clean, business-ready insights. The project uses two Kaggle retail datasets and is being developed as a professional portfolio project covering data cleaning, SQL analytics, dashboarding, and forecasting.

The goal is to answer practical retail business questions:

- Which products, stores, and customer segments drive revenue?
- How do sales patterns change over time?
- Which features help explain or forecast retail performance?
- How can cleaned data support dashboards and decision-making?

## Data Policy

Large datasets are not committed to GitHub.

The `data/` directory is ignored by Git to keep the repository lightweight and reproducible. Raw Kaggle downloads and cleaned outputs should stay on the local machine.

Expected local layout:

```text
data/
|-- raw/          # Original Kaggle files
|-- processed/    # Cleaned outputs generated by scripts
`-- external/     # Optional external reference data
```

## Data Cleaning

The first transformation step is implemented in:

```text
src/transform.py
```

It currently:

- Removes duplicate rows.
- Handles missing and incomplete records.
- Removes invalid business records such as cancelled invoices and negative sales.
- Standardizes key date and text fields.
- Creates cleaned local outputs in `data/processed/`.
- Generates a local data quality report.

Run the cleaning script:

```bash
python src/transform.py
```

## How to Reproduce

1. Clone the repository.
2. Create a Python virtual environment.
3. Install dependencies.
4. Download the Kaggle datasets locally into `data/raw/`.
5. Run the cleaning script.

```bash
pip install -r requirements.txt
python src/transform.py
```
